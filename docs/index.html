Auto Research

Auto Research is a reusable workflow system for coding agents. It keeps the stable Claude /autoresearch command family, adds a root-centered Codex skill bundle, and treats the repository as a shared source of truth instead of a single-runtime package.

Compatibility

Claude command names stay stable. Result logs preserve both research-results.tsv and autoresearch-results.tsv so older workflows keep working.

Core Loop

Baseline once, make one focused experiment, verify mechanically, keep strict improvements, discard regressions, record the outcome, and continue until the stop condition is met.
