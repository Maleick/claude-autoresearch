Subagent-First Autonomous Iteration — OpenCode

Auto Research

Auto Research is a subagent-first autonomous iteration engine for OpenCode. It keeps the stable /autoresearch command family, adds mode-specific workflows, and ships as an npm package.

Node.js ESM. TypeScript runtime. No Python in the shipped artifact.

Core Loop

Baseline once, make one focused experiment, verify mechanically, keep strict improvements, discard regressions, record the outcome, and continue until the stop condition is met.
